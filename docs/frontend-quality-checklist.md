# Frontend Quality Checklist

Use this when revisiting the social media demo.

| Area | Check |
| --- | --- |
| Feed | Empty feed, long post text, optimistic reaction behavior |
| Composer | Empty submission, character limits, image placeholder handling |
| Profile | Missing avatar, long names, mobile layout |
| Accessibility | Keyboard focus, button labels, form labels |
| Verification | Add React build/test command and screenshots for core flows |

Best next technical step: add component tests for post creation and reaction state changes.
